# Установка базовой системы
- Если стоит цель воспроизвести мою инфраструктуру, то на ПК необходимо поставить какой-то образ Linux. Это может быть любой современный дистрибутив, но я выбрал Arch Linux.

- Если вы хотите тоже установить Arch Linux, то рекомендую ознакомитсья с парой видео на ютубе. Установка через сприпт archinstall не такая сложная (советы по установке ниже созданы LLM, за их работоспособность не ручаюсь)


## Этап 1: Подготовка

### 1. Загрузка образа

Перейдите на [официальный сайт](https://archlinux.org/download/) и скачайте ISO-образ через торрент или зеркало. Рекомендуется также скачать файл подписи (`.sig`) для проверки целостности.

### 2. Запись на флешку

- **В Linux/macOS:** Используйте команду `dd` (замените `sdX` на ваше устройство):
```
sudo dd bs=4M if=path/to/archlinux.iso of=/dev/sdX conv=fsync oflag=direct status=progress
```
- **В Windows:** Используйте **Rufus** (режим ISO) или **BalenaEtcher**.


## Этап 2: Начальная настройка (Live-окружение)

### 1. Загрузка и сеть

Загрузитесь с флешки (обычно через F12 или F2 при старте). После появления командной строки подключите Wi-Fi с помощью утилиты `iwctl`:


```
iwctl
# Внутри iwctl:
device list                           # Узнать имя интерфейса (например, wlan0)
station wlan0 scan                    # Поиск сетей
station wlan0 get-networks            # Список сетей
station wlan0 connect SSID            # Подключение (введите пароль)
exit
```

Проверьте связь: `ping google.com`.

### 2. Обновление системных часов


```
timedatectl set-ntp true
```


## Этап 3: Работа с дисками

Используйте `cfdisk` для разметки. Для современной системы (UEFI) выберите тип **GPT**.

- **Раздел EFI:** ~512 МБ (тип: EFI System).
    
- **Раздел Swap:** Рекомендую 4–8 ГБ (тип: Linux swap).
    
- **Корневой раздел (Root):** Всё остальное место (тип: Linux filesystem).
    

**Форматирование:**

```
mkfs.fat -F 32 /dev/sdX1      # EFI раздел
mkswap /dev/sdX2              # Swap
swapon /dev/sdX2
mkfs.ext4 /dev/sdX3           # Root раздел
```

**Монтирование:**


```
mount /dev/sdX3 /mnt
mount --mkdir /dev/sdX1 /mnt/boot
```


## Этап 4: Установка базовой системы

Используем скрипт `pacstrap` для установки ядра и необходимых утилит:


```
pacstrap /mnt base linux linux-firmware base-devel nano networkmanager
```



## Этап 5: Настройка системы

### 1. Генерация fstab


```
genfstab -U /mnt >> /mnt/etc/fstab
```

### 2. Вход в систему (Chroot)


```
arch-chroot /mnt
```

### 3. Локализация и время


```
ln -sf /usr/share/zoneinfo/Europe/Moscow /etc/localtime
hwclock --systohc

# Раскомментируйте en_US.UTF-8 UTF-8 и ru_RU.UTF-8 UTF-8 в /etc/locale.gen
nano /etc/locale.gen
locale-gen
echo "LANG=ru_RU.UTF-8" > /etc/locale.conf
```

### 4. Сеть и пользователи


```
echo "arch-pc" > /etc/hostname
systemctl enable NetworkManager

passwd                         # Пароль root
useradd -m -G wheel artem      # Создание вашего пользователя
passwd artem                   # Пароль пользователя

# Разрешите группе wheel использовать sudo
EDITOR=nano visudo             # Раскомментируйте строку %wheel ALL=(ALL:ALL) ALL
```

### 5. Загрузчик (GRUB)


```
pacman -S grub efibootmgr
grub-install --target=x86_64-efi --efi-directory=/boot --bootloader-id=GRUB
grub-mkconfig -o /boot/grub/grub.cfg
```


## Этап 6: Установка графического окружения (XFCE)

Теперь установим видеодрайверы и саму оболочку.

### 1. Видеодрайверы

- **Intel:** `xf86-video-intel`
    
- **AMD:** `xf86-video-amdgpu`
    
- **Nvidia:** `nvidia nvidia-utils`
    

### 2. XFCE и экранный менеджер

```
pacman -S xorg xfce4 xfce4-goodies lightdm lightdm-gtk-greeter
systemctl enable lightdm
```

## Завершение

Выйдите из chroot, отмонтируйте диски и перезагрузитесь:

```
exit
umount -R /mnt
reboot
```

После перезагрузки у вас будет чистая система с XFCE. Для управления Wi-Fi в графическом интерфейсе используйте иконку NetworkManager в трее.
